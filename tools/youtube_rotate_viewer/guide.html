📋 사용방법

  1. 페이지 접속: YouTube Rotate Viewer 페이지에 접속합니다
  2. URL 입력: YouTube URL 입력란에 원하는 유튜브 영상 URL을 입력합니다
  3. 적용: "적용" 버튼을 클릭하거나 Enter 키를 누릅니다
  4. 회전: "R" 버튼을 클릭하여 영상을 90도씩 회전시킵니다
  5. 크기 조절: URL 파라미터로 크기를 조절할 수 있습니다

✨ 주요 기능

  • 🔄 영상 회전: 90도씩 회전하여 세로 화면에 최적화
  • 📏 크기 조절: 다양한 화면 크기에 맞춘 조절
  • 🎬 다양한 URL 지원: 일반 URL, 단축 URL, 임베드 URL 모두 지원
  • ⌨️ 키보드 지원: Enter 키로 빠른 적용
  • 🛡️ 에러 처리: 잘못된 URL 입력 시 친절한 안내
  • 🌙 다크 모드: 자동 다크 모드 지원

🎯 지원하는 YouTube URL 형식

  • 일반 URL: https://www.youtube.com/watch?v=KZAKBnyvnlw
  • 단축 URL: https://youtu.be/KZAKBnyvnlw
  • 임베드 URL: https://www.youtube.com/embed/KZAKBnyvnlw
  • 파라미터 포함: https://www.youtube.com/watch?v=KZAKBnyvnlw&t=30s

📐 URL 파라미터 사용법

파라미터 설명 예시
r 회전 각도 (u=0°, r=90°, d=180°, l=270°) ?r=r (90도 회전)
s 영상 크기 (픽셀 단위) ?s=800 (800px 크기)
u YouTube 영상 ID ?u=dQw4w9WgXcQ

🎯 사용 예시

1. 기본 사용

입력: https://youtu.be/KZAKBnyvnlw

결과: 해당 영상이 로드되고 "R" 버튼으로 회전 가능

2. 파라미터를 사용한 고급 설정

URL: 90도 회전된 800px 크기 영상

localhost:8000/tools/youtube_rotate_viewer/?r=r&s=800&u=dQw4w9WgXcQ

3. 다양한 회전 각도

  • ?r=u: 0도 (기본)
  • ?r=r: 90도 회전 (오른쪽)
  • ?r=d: 180도 회전 (뒤집기)
  • ?r=l: 270도 회전 (왼쪽)

💡 활용 팁

  • 세로 모니터: 세로 모니터에서 유튜브 영상을 풀스크린으로 시청
  • 모바일 세로 모드: 세로 모드에서 가로 영상을 편안하게 시청
  • 특수 화면: 회전된 화면이나 특수한 디스플레이에서 사용
  • 교육 목적: 프레젠테이션이나 강의에서 화면 회전 필요 시
  • 북마크 활용: 자주 사용하는 설정은 URL로 북마크

⚠️ 주의사항

  • YouTube의 정책에 따라 일부 영상은 임베드가 제한될 수 있습니다
  • 너무 큰 크기 설정 시 화면을 벗어날 수 있습니다
  • 회전 시 비율이 유지되므로 화면 크기를 고려하세요
  • 인터넷 연결이 필요하며, YouTube 서버 상태에 의존합니다

🔧 기술 정보

  • iframe 사용: YouTube 임베드 API를 활용
  • CSS Transform: CSS3 transform을 이용한 회전
  • URL 파싱: 다양한 YouTube URL 형식 자동 인식
  • 반응형 계산: 회전 시 크기 자동 조절
  • 에러 핸들링: 잘못된 URL 입력 시 사용자 친화적 안내
