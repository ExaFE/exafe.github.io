📝 Title Changer 사용방법

브라우저 제목과 화면 내용을 자유롭게 변경!

작업표시줄에서 쉽게 구분하거나 가상 데스크톱에서 화면을 빠르게 인식하세요

지금 사용해보기

📋 사용방법

  1. 페이지 접속: Title Changer 페이지에 접속합니다
  2. 텍스트 입력: "Text" 입력란에 원하는 제목을 입력합니다
  3. 폰트 크기 설정: "FontSize" 입력란에 원하는 크기를 입력합니다 (10-1000)
  4. Body 표시 선택: "ShowBody" 체크박스를 선택하면 화면에 큰 텍스트가 표시됩니다
  5. 적용: "Update" 버튼을 클릭하거나 Enter 키를 누릅니다

✨ 주요 기능

  • 🏷️ 브라우저 제목 변경: 작업표시줄에서 쉽게 구분 가능
  • 📺 화면 내용 표시: 가상 데스크톱에서 빠른 인식
  • 🔤 폰트 크기 조절: 10px부터 1000px까지 자유롭게 설정
  • 💾 URL 저장: 설정이 URL에 저장되어 북마크 가능
  • ⌨️ 키보드 지원: Enter 키로 빠른 적용
  • 🌙 다크 모드: 자동 다크 모드 지원

🎯 사용 예시

1. 브라우저 제목만 변경

입력: Text = "회의 자료", ShowBody = 체크 해제

결과: 브라우저 탭 제목이 "회의 자료"로 변경됩니다

예시 URL: localhost:8000/tools/title_body_changer/?text=회의%20자료

2. 화면에 큰 텍스트 표시

입력: Text = "프레젠테이션", FontSize = 500, ShowBody = 체크

결과: 화면에 500px 크기의 "프레젠테이션" 텍스트가 표시됩니다

예시 URL: localhost:8000/tools/title_body_changer/?show_body=true&font_size=500&text=프레젠테이션

💡 활용 팁

  • 멀티 모니터 환경: 각 모니터의 용도를 쉽게 구분
  • 가상 데스크톱: 데스크톱 전환 시 빠른 화면 인식
  • 회의/프레젠테이션: 화면 공유 시 목적 표시
  • 작업 분류: 프로젝트별로 브라우저 탭 구분
  • 알림 메시지: 중요한 메시지를 큰 글씨로 표시

⚠️ 주의사항

  • 폰트 크기가 너무 크면 화면을 벗어날 수 있습니다
  • URL에 한글이 포함되면 인코딩되어 복잡해 보일 수 있습니다
  • 브라우저를 새로고침하면 원래 제목으로 돌아갑니다
  • 일부 브라우저에서는 제목 변경이 제한될 수 있습니다

🔧 기술 정보

  • URL 파라미터: 설정 값을 URL에 저장하여 북마크 가능
  • 동적 스타일링: JavaScript로 실시간 폰트 크기 변경
  • 반응형 입력: Enter 키와 버튼 클릭 모두 지원
  • 자동 포커스: 페이지 로드 시 입력창에 자동 포커스
