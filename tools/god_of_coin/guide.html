🪙

결정 장애를 위한 완벽한 솔루션!

Yes 냐 No 냐, 그것이 문제로다!

지금 사용해보기

📋 사용방법

  1. 페이지 접속: 동전의 신 페이지에 접속합니다
  2. 화면 클릭: 화면 아무 곳이나 클릭하면 동전이 던져집니다
  3. 결과 확인: YES 또는 NO 결과가 나타납니다
  4. 점수 확인: 상단에 Yes/No 점수가 기록됩니다
  5. 다시 던지기: 결과가 나온 후 다시 클릭하면 새로운 동전 던지기가 시작됩니다

✨ 주요 기능

  • 🎯 간단한 조작: 화면 클릭만으로 동전 던지기
  • 📊 점수 기록: Yes/No 결과 자동 카운트
  • 🎨 시각적 효과: 아름다운 글로우 애니메이션
  • 📱 반응형 디자인: 모든 기기에서 완벽한 경험
  • 🌙 다크 모드: 자동 다크 모드 지원
  • 🔄 무한 사용: 원하는 만큼 반복 사용 가능

🎯 사용 팁

  • 결정이 어려울 때: 중요한 선택을 앞두고 고민될 때 사용하세요
  • 게임이나 놀이: 친구들과 함께 재미있는 게임으로 활용하세요
  • 일상의 소소한 선택: 메뉴 선택, 활동 결정 등에 활용하세요
  • 점수 활용: 여러 번 던져서 통계적 결과를 확인해보세요

🔧 기술 정보

  • 랜덤 알고리즘: JavaScript의 Math.random()을 기반으로 한 공정한 결과
  • 애니메이션: CSS3 애니메이션으로 부드러운 시각적 효과
  • 반응형: 모든 화면 크기에 최적화된 레이아웃
  • 접근성: 키보드 네비게이션 및 스크린 리더 지원
